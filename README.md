# DataToolkit

DataToolkit is a command-line application built with Python to practice fundamental programming concepts and clean code.

## Features

### Text Tools

- Character Counter
- Word Counter
- Uppercase
- Lowercase
- Title Case
- Reverse Text
- Find & Replace
- Remove Duplicate Line
- Sort Lines

## Requirements

Python 3.14+

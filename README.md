# DataToolkit

DataToolkit is a command-line application built with Python to demonstrate fundamental Python programming skills, clean code practices, and modular project architecture.

This project is developed as a personal learning portfolio while studying software engineering. The goal is not to create the largest toolkit, but to build every feature with a strong understanding of Python fundamentals.

## 🚀 Getting Started

Clone the repository.

```bash
https://github.com/Sendyidrwsta/DataToolkit.git
```

Go to the project directory.

```bash
cd DataToolkit
```

Run the application.

```bash
python src/main.py
```
